### Dependencies

Make sure that you have the following modules:  
`os.system`  
`tkinter`  
`tkinter.messagebox`  
`tkinter.simpledialog`  
`termcolor.cprint`  
`random`

### How to run
  Run the `main.py` file to start. If there are errors, please submit a error on our [sourceforge](https://sourceforge.net/p/supersku-v1)
